# My Portfolio Project

## About this project
A personal portfolio website for Alex Kastanis, built with plain 
HTML and CSS. No build tools or frameworks.

## Files
- `index.html` — the homepage with About Me, Skills, and Contact sections
- `styles.css` — all styling (uses CSS variables for theming)
- `README.md` — GitHub-facing description
- `.gitignore` — standard OS and env excludes

## Coding preferences
- Use 2 spaces for indentation
- Keep HTML semantic (use `<section>`, `<article>`, proper headings)
- Use CSS variables for all colors to support future theming
- Mobile-first responsive design
- Don't add JavaScript unless I explicitly ask for it

## Style preferences
- Warm color palette (terracotta, cream, deep brown)
- Clean, modern typography
- Subtle hover effects, no aggressive animations

## What NOT to edit
- `.git/` folder (Git's business)
- Don't modify `.gitignore` unless I ask
